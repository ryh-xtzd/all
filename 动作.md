# 胸

## 蝴蝶机夹胸

1. 手握把位置和乳头平行，握住中段，手臂微弯
2. 全握，手掌偏下位置发力，想象手掌推过来
3. 想像上臂靠近胸肌，离胸越近越好
4. 肩胛骨内收挺胸，稍微反弓

## 坐姿推胸

1. 手在中胸位置，即乳头偏上方一点
2. 握把调节的越低对胸部的拉伸越好
3. 向前推手臂不要打太直，肘关节运动向后，小臂平行地面，不要抬高
4. 肩胛骨内收挺胸，稍微反弓

## 杠铃卧推

1. 脚掌瞪地面，脚尖略微朝外，脚往屁股方向一点
2. 肩胛骨内收
3. 眼镜在杆下方
4. 找到手掌心顶住杆的感觉，肘关节45度左右

## 上斜推胸

1. 扶手最低点在胸的二分之一位置
2. 身体完全靠住靠板，肩胛骨后收顶住靠板，略微反弓
3. 拇指按在顶端保持大拇指伸直的位置就是合适的位置
4. 肘关节和身体角度在45度左右
5. 运动行程不一定落到底，顶底肘略微弯曲

## 俯卧撑

1. 手与肩部同宽，腿与跨同宽，核心收紧

# 背

## 引体向上

1. 半握1.5肩宽
2. 掰杆肩关节外旋
3. 拉起时，小臂垂直地面身体略微后仰
4. 慢下方

## 高位下拉

1. 半握1.5肩宽
2. 下拉到下巴高度
3. 肘关节略微内收向下，指向地面
4. 身体保持中立，可以随着重量下移是向后倾斜，向上还原
5. 可以适当外旋（与引体相反）

## 坐姿划船

1. 手竖直，膝盖略微弯曲，靠后坐
2. 身体挺直，挺胸不是反弓，向前后期可以含胸一点点
3. 向后拉向肚脐，身体挺胸可以整体向后一点
4. 注意力放在肘关节向后拉到身体旁边

# 肩后束

## 蝴蝶机反向飞鸟

1. 手掌握把比肩膀略微低一点
2. 拇指握住握把顶端，手掌靠住杆子，也可以靠前这样给肘关节更靠外
3. 上臂为轴，肩膀平了就行不用再向后了
4. 略微后座，身体前倾，趴在板上
5. 肘关节固定，画半圆

## 哑铃俯身飞鸟

1. 屁股向前，双脚合并，趴下来，胸到大腿
2. 手自然下垂，手肘角度保持，保持沉肩 
3. 横向两侧来回，不要向前和向后
4. 想象手掌带起哑铃，哑铃最后垂直于地面，手掌外侧向天花板拉起，上臂和身体在一个平面

## 绳索面拉

1. 反握
2. 略微向后，手臂伸直之后重量片会略微起来，重心在脚后跟 
3. 两种站姿，第一种髋关节稍微向后坐，重心向下压和向后移，第二种挺胸向后拉时身体略微后仰
4. 运动轨迹，第一种，横着向后拉确保掌心朝下，第二种，保持上臂与地面平行，绳子向上扬一点，打开绳子外翻掌心相对，手臂和背一个平面

# 肱二头

## 龙门架弯举

1. 器械最底端
2. 手臂垂直地面
3. 向天花板拉起

# 腿

## 俯身腿弯举

1. 靠板卡在脚踝位置，俯身下来膝盖放在屁股板下方，身体放松，手抓握把，屁股不要接力
2. 膝盖与髋同宽，脚尖略微发力勾一点
3. 发力脚后跟向上题，向上时小腿垂直再往里一点，向下可以伸直
4. 颈椎向上略微抬高，保证胸部离开靠板一点，便于呼吸，腰腹部压紧靠板

## 坐姿内收

1. 脚掌不完全踩在踏板上，脚尖抬高，大腿膝盖发力，抓好握把 
2. 屁股顶住靠板，不用保持背停止，身体可以前倾
3. 快收慢开

## 箭步蹲

1. 双腿与跨同宽
2. 迈开后脚尖抬起，前后最好90度，但膝盖低不要着地
3. 迈开，蹲下，站直，收回
4. 竖着向下蹲，迈开不要着急，等稳定了再进行运动

# 肩前束

## 器械推举

1. 把手与下巴持平，掌跟握住把手不是掌心，1.5倍宽
2. 背部贴紧，下落到把手和嘴巴持平，小臂垂直地面

## 坐姿推肩

1. 小臂垂直地面

# 肩中束

## 龙门架侧平举

1. 绳索膝盖位置，屁股可以后撅一点
2. 10-15度倾斜
3. 手腕不动，想象上臂发力，与肩部平行即可

## 侧平举

1. 哑铃靠在腿前侧一点，可以半握
2. 肘关节微微弯曲，身体前倾
3. 顶点保持1s

# 肱三头

## 绳索臂屈伸

1. 上臂不要动，重量大身体可以前倾，但挺胸，夹住手臂，不要耸肩
2. 抬到上臂要去移动就停止
3. 翻转手掌会练外侧，如果找到掌心相对的感觉就会练长头

# 屁股

## 坐姿外扩

1. 屁股顶住靠板，不用保持背停止，身体可以前倾

## 臀桥

1. 杆子对准髋关节前侧
2. 双脚与髋同宽，脚尖略微向外，踩实踏板，后背紧贴靠背、
3. 下方至髋部夹角为90度，向上至屁股收紧，脚跟稳定腿不要发力
4. 最高点肩，髋，膝成一条直线，不要快速下落，顶点可以屁股发力保持
